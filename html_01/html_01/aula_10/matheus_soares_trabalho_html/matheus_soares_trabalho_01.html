1 - O que é HTML ?

O HTML é a abrevição em inglês de "Hypertext markup Language" e traduzido para o português é "Linguagem de marcação de hipertexto". Ele serve para fazer a estrutura básica de um site e também ajuda no desenvolvimento.

Os hipertexto são links que conectam sites uns com os outros, e pode ser vista de várias formas atráves de palavras e até vídeos.

2 - O que é um elemento HTML? Dê exemplos.

Os elementos é aonde define a estrutura e o conteúdo do documento. Ele é aonde se desenvolve a criação de links, texto, imagens e entre outros.

<h1> Títulos </h1>
<a> Links </a>
<p> Parágrafos <p>

3 - Que ferramentas são necessárias para construir um site HTML?

É preciso de editor de texto - Visual studio code. E ter uma estrutura base para a criação do site.

<!DOCTYPE html>
<html lang="pt">
<head>
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>
</head>
<body>
</body>
</html>

5 - O elemento "br" necessita ser fechado?

Ele faz parte do "void element". No void element não é preciso ter uma tag de encerramento e nele também não é possível ter conteúdo, e só podem ter atributos. Então o elemento "br" não necessita ser fechado.

6 - Para que servem os atributos? Dê um exemplo.

Os atributos são os que modificam os elementos. Nele pode específicar melhor os elementos, e modificá-los de várias formas como por os texto diferentes, cores, bordas, etc.

<p style="border: 2px solid black; width: 500px; height: 500px;">
